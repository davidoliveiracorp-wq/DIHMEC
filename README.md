# Aplicação Lucid React

Aplicação React com Lucid React para criação de diagramas e visualizações.

## Instalação

1. Instale as dependências:
```bash
npm install
```

## Execução

### Modo de Desenvolvimento
```bash
npm run dev
```
Isso iniciará o servidor de desenvolvimento Vite na porta 3000.

### Build para Produção
```bash
npm run build
```
Isso criará os arquivos otimizados na pasta `dist/`.

### Preview do Build
```bash
npm run preview
```

## Estrutura de Arquivos

- `index.jsx` - Componente principal React com diagrama Lucid React
- `index-react.html` - Arquivo HTML de entrada
- `vite.config.js` - Configuração do Vite
- `package.json` - Dependências do projeto

## Funcionalidades

O diagrama inclui:
- Retângulos com texto
- Círculos
- Elipses
- Linhas com setas
- Grupos de elementos
- Paths personalizados (curvas)

## Personalização

Você pode modificar o arquivo `index.jsx` para criar seus próprios diagramas usando os componentes do Lucid React.
